# 现金流量表重构指南（现金流量分析表）

## 概述

本模块将传统现金流量表重构为"现金流量分析表"，分析自由现金流、资本支出、债务筹资等核心现金流指标，便于评估企业的现金创造能力和投资活动。

## 重构原理

### 传统现金流量表 vs 重构后的现金流量表

**传统结构**：
```
经营活动现金流量
投资活动现金流量
筹资活动现金流量
```

**重构后结构（现金流量分析表）**：
```
自由现金流量分析
  - 收入现金含量
  - 成本费用付现率
  - 利润现金含量

长期经营资产投资活动分析
  - 长期经营资产净投资额
  - 长期经营资产扩张性资本支出

并购活动分析
  - 净合并额

资本支出分析
  - 资本支出总额
  - 扩张性资本支出

债务筹资分析
  - 债务筹资净额
```

### 核心区别

1. **现金流质量分析**
   - 收入现金含量：销售收到现金 / 营业收入
   - 利润现金含量：经营现金流 / 净利润

2. **资本支出分类**
   - 维持性资本支出：用于维持现有经营规模的支出（折旧摊销）
   - 扩张性资本支出：用于扩大经营规模的支出

3. **自由现金流**
   - FCFF：企业自由现金流（归属于所有投资者）
   - FCFE：股权自由现金流（归属于股东）

## 重构规则详细说明

### 结构设计

```
现金流量分析表
├── 自由现金流量分析
│   ├── 销售商品、提供劳务收到的现金
│   ├── 营业收入
│   ├── 口径一收入现金含量 = 销售收到现金 / 营业收入
│   ├── 购买商品、接收劳务支付的现金
│   ├── 支付给职工及为职工支付的现金
│   ├── 营业总成本
│   ├── 成本费用付现率 = (购买支付现金 / 1.17 + 职工支付现金) / 营业总成本
│   ├── 经营活动产生的现金流量净额
│   ├── 息前税后经营利润
│   ├── 息前税后经营利润现金含量 = 经营现金流 / 息前税后经营利润
│   ├── 净利润
│   └── 净利润现金含量 = 经营现金流 / 净利润
├── 经营活动产生的现金流量净额分析
│   ├── 经营活动产生的现金流量净额
│   ├── 加：资产减值准备
│   ├── 固定资产折旧、油气资产折耗、生产性生物资产折旧
│   ├── 无形资产摊销
│   ├── 长期待摊费用摊销
│   ├── 处置固定资产、无形资产和其他长期资产的损失
│   ├── 固定资产报废损失
│   ├── 非付现成本费用合计
│   └── 经营资产自由现金流量 = 经营现金流 - 非付现成本费用
├── 投资活动产生的现金流量分析
│   ├── 收回投资收到的现金
│   ├── 取得投资收益所收到的现金
│   ├── 处置固定资产、无形资产及其他长期资产收到的现金
│   ├── 处置子公司、合营联营企业及其他营业单位收到的现金净额
│   ├── 收到的其他与投资活动相关的现金
│   ├── 投资活动现金流入小计
│   ├── 购建固定资产、无形资产及其他长期资产所支付的现金
│   ├── 投资所支付的现金
│   ├── 取得子公司、合营联营企业及其他营业单位支付的现金净额
│   ├── 支付的其他与投资活动有关的现金
│   ├── 投资活动现金流出小计
│   └── 投资活动产生的现金流净额 = 流入小计 - 流出小计
├── 长期经营资产投资活动现金流量分析
│   ├── 处置固定资产、无形资产及其他长期资产收到的现金
│   ├── 购建固定资产、无形资产及其他长期资产所支付的现金
│   ├── 长期经营资产净投资额 = 购建支付 - 处置收到
│   ├── 固定资产折旧、油气资产折耗、生产性生物资产折旧
│   ├── 无形资产摊销
│   ├── 长期待摊费用摊销
│   ├── 处置固定资产、无形资产和其他长期资产的损失
│   ├── 固定资产报废损失
│   ├── 长期经营资产扩张性资本支出 = 净投资额 - 折旧 - 摊销 - 处置损失 - 报废损失
│   ├── 长期经营资产合计（从资产负债表获取）
│   └── 长期经营资产扩张性资本支出比例 = 扩张性资本支出 / 期初长期经营资产
├── 并购活动现金流量分析
│   ├── 处置子公司、合营联营企业及其他营业单位收到的现金净额
│   ├── 取得子公司、合营联营企业及其他营业单位支付的现金净额
│   └── 净合并额 = 取得支付 - 处置收到
├── 资本支出分析
│   ├── 长期经营资产净投资额
│   ├── 长期经营资产扩张性资本支出
│   ├── 净合并额
│   ├── 资本支出总额 = 长期经营资产净投资额 + 净合并额
│   ├── 扩张性资本支出 = 长期经营资产扩张性资本支出 + 净合并额
│   └── 扩张性资本支出占长期资产期初净额的比例 = 扩张性资本支出 / 期初长期经营资产
└── 债务筹资现金流量分析
    ├── 取得借款收到的现金
    ├── 发行债券收到的现金
    ├── 偿付债务支付的现金
    ├── 偿付利息支付的现金
    └── 债务筹资净额 = 取得借款 + 发行债券 - 偿付债务 - 偿付利息
```

### 核心计算公式

#### 自由现金流量分析

| 项目 | 计算规则 |
|------|---------|
| 口径一收入现金含量 | = 销售商品、提供劳务收到的现金 / 营业收入 |
| 成本费用付现率 | = (购买商品、接收劳务支付的现金 / 1.17 + 支付给职工及为职工支付的现金) / 营业总成本 |
| 息前税后经营利润现金含量 | = 经营活动产生的现金流量净额 / 息前税后经营利润 |
| 净利润现金含量 | = 经营活动产生的现金流量净额 / 净利润 |

> **注意**：
> - 购买支付现金除以1.17是为了剔除增值税影响
> - 营业收入和营业总成本从利润表获取
> - 息前税后经营利润从重构后的利润表获取

#### 经营活动现金流量净额分析

| 项目 | 计算规则 |
|------|---------|
| 非付现成本费用合计 | = 资产减值准备 + 固定资产折旧 + 无形资产摊销 + 长期待摊费用摊销 + 处置损失 + 报废损失 |
| 经营资产自由现金流量 | = 经营活动产生的现金流量净额 - 非付现成本费用合计 |

#### 长期经营资产投资活动分析

| 项目 | 计算规则 |
|------|---------|
| 长期经营资产净投资额 | = 购建固定资产、无形资产及其他长期资产所支付的现金 - 处置固定资产、无形资产及其他长期资产收到的现金 |
| **长期经营资产扩张性资本支出** | = 长期经营资产净投资额 - 固定资产折旧 - 无形资产摊销 - 长期待摊费用摊销 - 处置损失 - 报废损失 |
| 长期经营资产扩张性资本支出比例 | = 长期经营资产扩张性资本支出 / 期初长期经营资产合计 |

> **注意**：
> - 长期经营资产合计从资产负债表获取
> - 期初值 = 上一期的期末值
> - 折旧、摊销、损失均取绝对值

#### 并购活动分析

| 项目 | 计算规则 |
|------|---------|
| 净合并额 | = 取得子公司、合营联营企业及其他营业单位支付的现金净额 - 处置子公司、合营联营企业及其他营业单位收到的现金净额 |

#### 资本支出分析

| 项目 | 计算规则 |
|------|---------|
| **资本支出总额** | = 长期经营资产净投资额 + 净合并额 |
| **扩张性资本支出** | = 长期经营资产扩张性资本支出 + 净合并额 |
| 扩张性资本支出占长期资产期初净额的比例 | = 扩张性资本支出 / 期初长期经营资产合计 |

> **重要**：
> - 资本支出总额 = 长期经营资产净投资额 + 净合并额
> - 扩张性资本支出 = 长期经营资产扩张性资本支出 + 净合并额
> - 这两个指标都包含了并购活动的影响

#### 债务筹资分析

| 项目 | 计算规则 |
|------|---------|
| 债务筹资净额 | = 取得借款收到的现金 + 发行债券收到的现金 - 偿付债务支付的现金 - 偿付利息支付的现金 |

### 折旧及摊销合计详细说明

**折旧及摊销合计**包含以下5个项目：

```
折旧及摊销合计 = 固定资产折旧、油气资产折耗、生产性生物资产折旧
               + 无形资产摊销
               + 长期待摊费用摊销
               + 处置固定资产、无形资产和其他长期资产的损失
               + 固定资产报废损失
```

这个合计在以下场景中使用：
1. 计算长期经营资产扩张性资本支出
2. 计算FCFF（企业自由现金流）
3. 计算FCFE（股权自由现金流）

### 自由现金流计算

#### FCFF（企业自由现金流）

```
FCFF = 息税前经营利润税后 + 折旧及摊销合计 - 资本支出总额 - 营运资本变化量

其中：
息税前经营利润税后 = 息税前经营利润 × (1 - 实际所得税税率)
营运资本变化量 = 当期周转性经营投入合计 - 上期周转性经营投入合计
```

#### FCFE（股权自由现金流）

```
FCFE = 净利润 + 折旧及摊销合计 - 资本支出总额 - 营运资本变化量 + 债务变化

其中：
债务变化 = 当期有息债务合计 - 上期有息债务合计
```

> **注意**：
> - FCFF和FCFE的计算在HTML报告生成器中实现
> - 这里的现金流量表重构主要提供基础数据

## 使用方法

### 1. 基本使用

```python
import pandas as pd
from cashflow_statement_restructure import restructure_cashflow_statement

# 读取现金流量表数据（转置格式）
df_cashflow = pd.read_csv('data/603345.SH_cashflow.csv', encoding='utf-8-sig')
df_balance = pd.read_csv('data/603345.SH_balancesheet_restructured.csv', encoding='utf-8-sig')
df_income = pd.read_csv('data/603345.SH_income_restructured.csv', encoding='utf-8-sig')

# 重构现金流量表（需要资产负债表和利润表数据）
df_restructured = restructure_cashflow_statement(df_cashflow, df_balance, df_income)

# 保存结果
df_restructured.to_csv('data/603345.SH_cashflow_restructured.csv', 
                       index=False, encoding='utf-8-sig')
```

### 2. 与TushareClient集成

```python
from tushare_client import TushareClient
from cashflow_statement_restructure import restructure_cashflow_statement

# 初始化客户端
client = TushareClient()

    cashflow, balance_restructured, income_restructured

# 保存

                   '603345.SH')
```

## 字段说明

### 现金流质量指标

| 项目 | 说明 | 理想值 |
|------|------|--------|
| 口径一收入现金含量 | 销售收到现金占营业收入的比例 | > 100% |
| 成本费用付现率 | 成本费用的付现比例 | 适中 |
| 息前税后经营利润现金含量 | 经营现金流占息前税后经营利润的比例 | > 100% |
| 净利润现金含量 | 经营现金流占净利润的比例 | > 100% |

### 资本支出指标

| 项目 | 说明 |
|------|------|
| 长期经营资产净投资额 | 长期资产投资的现金净流出 |
| 长期经营资产扩张性资本支出 | 用于扩大经营规模的资本支出 |
| 净合并额 | 并购活动的现金净流出 |
| 资本支出总额 | 总的资本支出（包含并购） |
| 扩张性资本支出 | 总的扩张性支出（包含并购） |

### 特殊调整项

1. **购买支付现金除以1.17**
   ```
   成本费用付现率 = (购买支付现金 / 1.17 + 职工支付现金) / 营业总成本
   ```
   - 购买商品支付的现金包含增值税，需要除以1.17剔除增值税影响

2. **折旧摊销取绝对值**
   ```
   长期经营资产扩张性资本支出 = 净投资额 - |折旧| - |摊销| - |损失|
   ```
   - 现金流量表中折旧摊销可能为负值，需要取绝对值

3. **期初长期经营资产**
   ```
   期初长期经营资产 = 上一期的期末长期经营资产
   ```
   - 从资产负债表的上一期数据获取

### 1. 现金流质量分析

```python
# 计算现金流质量指标

    print(f"净利润现金含量优秀: {净利润现金含量*100:.2f}%")
```

### 2. 资本支出分析


### 3. 自由现金流分析

```python
# 计算自由现金流（需要在HTML报告生成器中实现）
```

## 注意事项

1. **数据格式要求**
   - 输入数据必须是转置格式（字段名为行，日期为列）
   - 需要同时提供资产负债表和利润表的重构数据

2. **缺失值处理**
   - 缺失值会被自动填充为0进行计算
   - 建议在使用前检查数据完整性

3. **跨表数据引用**
   - 营业收入、营业总成本从利润表获取
   - 息前税后经营利润从重构后的利润表获取
   - 长期经营资产合计从重构后的资产负债表获取
   - 周转性经营投入合计从重构后的资产负债表获取

4. **期初期末数据**
   - 期初数据 = 上一期的期末数据
   - 第一期数据没有期初值，相关比率会设为None或0

5. **增值税处理**
   - 购买商品支付的现金包含增值税
   - 计算成本费用付现率时需要除以1.17剔除增值税
