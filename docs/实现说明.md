# 财务报表重构实现说明

## 1. "其他非流动资产"的归属决策

### 决策：严格按照文档规则，归入长期经营资产

根据`财务报表重构规则详细说明.md`第3.1.2节和3.1.4节，"其他非流动资产"应归入**长期经营资产合计**，而不是金融资产合计。

### 验证结果

#### 600900（长江电力）- ✅ 完全匹配
- **金融资产合计**: 181.86亿（与参考Excel完全一致）
- **Invested Capital**: 5035.71亿
- **差异**: 0.00亿

#### 000333（美的集团）- ⚠️ 部分差异
- **金融资产合计**: 2196.54亿（参考Excel: 3234.30亿）
- **差异**: 1037.76亿（主要是"其他非流动资产"921亿）
- **Invested Capital**: 708.15亿（参考Excel可能期望为负数）

### 差异原因

"其他非流动资产"的性质因公司而异：

1. **公用事业公司（如长江电力）**
   - 其他非流动资产主要是经营性质（工程物资、预付设备款等）
   - 应归入长期经营资产 ✅

2. **制造业公司（如美的集团）**
   - 其他非流动资产可能包含较多金融性质资产（理财产品、委托贷款等）
   - 参考Excel将其归入金融资产
   - 但按照会计准则的一般理解，仍应归入长期经营资产

### 实现方案

**当前实现（方案1）**：

### 优势
1. ✅ 严格遵循文档规则
2. ✅ 符合会计准则的一般理解
3. ✅ 对大多数公司（如600900）计算结果正确
4. ✅ 逻辑一致，不需要针对不同公司做特殊处理

### 局限性
1. ⚠️ 对于某些公司（如000333），可能与参考Excel存在差异
2. ⚠️ 需要用户理解"其他非流动资产"的实际构成可能因公司而异

## 2. 其他实现细节

### 2.1 默认参数
- `--annual-ttm`: 默认为True（自动生成年报+TTM数据）
- `--years`: 默认为None（覆盖所有历史数据）

### 2.2 数据单位
- 原始数据：元
- 重构文件：元（保持原始单位）
- HTML报告：亿元（除以1e8转换）

### 2.3 TTM计算
- 利润表和现金流量表：`今年Q累计 - 去年同Q累计 + 去年Q4累计`
- 资产负债表：使用最新季度数据

## 3. 使用方法

```bash
# 生成重构数据和年报+TTM（覆盖所有历史）
python3 main.py 000333

# 生成HTML报告
python3 example_html_report.py
```

## 5. 已知问题

### 5.1 字段名称差异
某些字段在参考Excel中有细分（如"发放贷款及垫款(流动)"和"发放贷款及垫款(非流动)"），但原始API数据中只有合并字段"发放贷款及垫款"。这会导致小幅差异。

### 5.2 "其他非流动资产"的归属
如前所述，不同公司的"其他非流动资产"性质不同，当前实现统一归入长期经营资产。

## 6. 未来改进方向

1. **动态判断"其他非流动资产"归属**
   - 可以根据公司行业、业务性质或配置文件来决定
   - 需要额外的规则引擎或人工配置

2. **支持附注数据**
   - 使用财务报表附注数据来细分某些合并字段
   - 需要额外的数据源

3. **行业特定规则**
   - 针对不同行业（制造业、金融业、公用事业等）使用不同的重构规则
   - 需要行业分类和规则库
