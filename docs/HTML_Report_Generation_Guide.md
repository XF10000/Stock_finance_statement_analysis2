# HTML财务分析报告生成技术文档

## 1. 概述

### 1.1 文档目的
本文档详细说明 `financial_report.html` 的生成逻辑、图表配置、指标计算公式和数据来源。**本文档与实际生成的HTML报告图表完全对应。**

### 1.2 核心文件
- **生成器**: `html_report_generator.py`
- **模板**: `templates/financial_report_template.html`
- **输出**: `data/{stock_code}_financial_report.html`

### 1.3 报告结构
HTML报告包含以下7个部分，共17个图表：

1. **资产负债表分析** - 4个图表
2. **利润表分析** - 3个图表
3. **经营效率分析** - 5个图表
4. **财务成本分析** - 1个图表
5. **长期资产投资和并购活动分析** - 1个图表
6. **投资收益分析** - 1个图表
7. **自由现金流分析** - 2个图表

---

## 2. 数据源说明

### 2.1 输入数据
报告生成需要以下CSV文件：
- `{stock_code}_balance_sheet_annual_ttm.csv` - 资产负债表年报+TTM（重构后）
- `{stock_code}_income_statement_annual_ttm.csv` - 利润表年报+TTM（重构后）
- `{stock_code}_cashflow_statement_annual_ttm.csv` - 现金流量表年报+TTM（重构后）

### 2.2 数据格式
所有CSV文件格式：
```
项目,2025Q3-TTM,2024/12/31,2023/12/31,...
营业收入,15421000000,15127000000,12168575000,...
净利润,1403000000,1515000000,1216857500,...
...
```

### 2.3 日期列识别
- 年报列格式：`YYYY/12/31` (如 `2024/12/31`)
- TTM列格式：`YYYYQN-TTM` (如 `2025Q3-TTM`)
- 按时间倒序排列（最新在左）

### 3.2 图表2：资本期限分析

**图表ID**: `chart_capital_structure`  
**图表类型**: 堆叠柱状图+折线图

**数据项**:
- **柱状图**:
  - 短期债务
  - 长期债务
  - 所有者权益合计
- **折线图**:
  - 有息债务率（计算字段）

**计算公式**:

#### 有息债务率
```
有息债务率 = 有息债务合计 / 资本总额 × 100%

其中：
有息债务合计 = 短期债务 + 长期债务（从重构后的资产负债表中直接读取）
资本总额 = 所有者权益合计 + 有息债务合计（从重构后的资产负债表中直接读取）
```

**数据来源**: 资产负债表（重构后）  
**单位**: 亿元（柱状图）、百分比（折线图）  
**说明**: 展示资本结构及有息债务占比

### 3.4 图表4：经营资本-营业收入比例

**图表ID**: `chart_working_capital_ratio`  
**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:

```
存货-营业收入比例 = 存货 / 营业收入 × 100%
```

**数据来源**: 资产负债表（周转性经营投入、存货）+ 利润表（营业收入）

### 4.3 图表3：营业费用

**图表ID**: `chart_operating_expenses`  
**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - 销售费用
  - 管理费用
  - 研发费用
  - 税金及附加
  - 资产减值损失
  - 信用减值损失
  - 营业外收入
  - 营业外支出
- **折线图**:
  - 销售费用率
  - 管理费用率
  - 研发费用率
  - 税金及附加率（计算字段）
  - 经营资产减值损失率（计算字段）
  - 营业外收支净额率（计算字段）
  - 总费用率（计算字段）

**计算公式**:

#### 税金及附加率
```
税金及附加率 = 税金及附加 / 营业收入 × 100%
```

#### 经营资产减值损失率
```
经营资产减值损失率 = (资产减值损失 + 信用减值损失) / 营业收入 × 100%
```

#### 营业外收支净额
```
营业外收支净额 = 营业外收入 - 营业外支出
```

#### 营业外收支净额率
```
营业外收支净额率 = 营业外收支净额 / 营业收入 × 100%
```

#### 总费用率
```
总费用率 = 税金及附加率 + 销售费用率 + 管理费用率 + 经营资产减值损失率 - 营业外收支净额率

注意：
1. 不包含研发费用率
2. 营业外收支净额率是减去的，不是加上
```

**数据来源**: 利润表（重构后）  
**单位**: 亿元（柱状图）、百分比（折线图）  
**说明**: 展示各项费用及费用率

## 5. 经营效率分析图表（5个）

### 5.1 图表1：ROIC

**图表ID**: `chart_roic`  
**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - Invested Capital（计算字段）
  - 息前税后经营利润
- **折线图**:
  - ROIC（计算字段）
  - 息前税后经营利润增速（计算字段）

**计算公式**:

#### Invested Capital（投入资本）
```
Invested Capital = 所有者权益合计 + 有息债务合计 - 金融资产合计
```

#### ROIC（投入资本回报率）
```
ROIC = 息前税后经营利润 / 平均投入资本 × 100%

其中：
平均投入资本 = (期初投入资本 + 期末投入资本) / 2

特殊处理：
- 第一期：使用期末投入资本（无期初数据）
- 其他期：使用平均投入资本
```

#### 息前税后经营利润增速
```
息前税后经营利润增速 = (当期息前税后经营利润 - 上期息前税后经营利润) / 上期息前税后经营利润 × 100%

特殊处理：
- 第一期：无增速数据
```

**数据来源**: 资产负债表（所有者权益、有息债务、金融资产）+ 利润表（息前税后经营利润）  
**单位**: 亿元（柱状图）、百分比（折线图）  
**说明**: 展示投入资本回报率及经营利润增速

---

### 5.2 图表2：Equity, Debt, ROE and ROIC

**图表ID**: `chart_roe_roic`  
**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - 有息债务合计
  - 所有者权益合计
- **折线图**:
  - ROIC（计算字段）
  - ROE（计算字段）

**计算公式**:

#### ROIC
```
ROIC = 息前税后经营利润 / 平均投入资本 × 100%
（计算方法同图表1）
```

#### ROE（净资产收益率）
```
ROE = 净利润 / 平均所有者权益合计 × 100%

其中：
平均所有者权益合计 = (期初所有者权益 + 期末所有者权益) / 2

特殊处理：
- 第一期：使用期末所有者权益（无期初数据）
```

**数据来源**: 资产负债表（有息债务、所有者权益）+ 利润表（净利润、息前税后经营利润）  
**单位**: 亿元（柱状图）、百分比（折线图）  
**说明**: 对比ROE和ROIC

### 5.3 图表3：经营资产效率

**图表ID**: `chart_asset_efficiency`  
**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - 净利润
- **折线图**:
  - 经营资产周转率（计算字段）
  - 长期经营资产周转率（计算字段）
  - 固定资产周转率（计算字段）

**计算公式**:

#### 经营资产周转率
```
经营资产周转率 = 营业收入 / 平均经营资产

其中：
经营资产 = 周转性经营投入合计 + 长期经营资产合计
平均经营资产 = (期初经营资产 + 期末经营资产) / 2
```

#### 长期经营资产周转率
```
长期经营资产周转率 = 营业收入 / 平均长期经营资产合计
```

#### 固定资产周转率
```
固定资产周转率 = 营业收入 / 平均固定资产
```

**数据来源**: 资产负债表（经营资产）+ 利润表（营业收入、净利润）  
**单位**: 亿元（柱状图）、次数（折线图）  
**说明**: 展示资产周转效率

### 5.4 图表4：周转天数及营业周期

**图表ID**: `chart_turnover_days`  
**图表类型**: 折线图

**数据项**:
- 应收账款周转天数（计算字段）
- 存货周转天数（计算字段）
- 应付账款周转天数（计算字段）
- 营业周期（计算字段）
- 现金周期（计算字段）

**计算公式**:

#### 应收账款周转天数
```
应收账款周转天数 = 365 / 应收账款周转率

其中：
应收账款周转率 = 营业收入 / 平均应收账款及票据
```

#### 存货周转天数
```
存货周转天数 = 365 / 存货周转率

其中：
存货周转率 = 营业成本 / 平均存货
```

#### 应付账款周转天数
```
应付账款周转天数 = 365 / 应付账款周转率

其中：
应付账款周转率 = 营业成本 / 平均应付账款及票据
```

#### 营业周期
```
营业周期 = 应收账款周转天数 + 存货周转天数
```

#### 现金周期
```
现金周期 = 营业周期 - 应付账款周转天数
```

**数据来源**: 资产负债表（应收账款、存货、应付账款）+ 利润表（营业收入、营业成本）  
**单位**: 天  
**说明**: 展示营运资金周转效率

## 7. 长期资产投资和并购活动分析图表（1个）

### 7.1 图表1：长期经营资产投资和并购活动分析

**图表ID**: `chart_capex_analysis`  
**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - 资本支出净额（计算字段）
  - 扩张性资本支出（计算字段）
  - 长期经营资产合计
- **折线图**:
  - 扩张性资本支出占长期资产期初净额的比例（计算字段）

**计算公式**:

#### 资本支出净额
```
资本支出净额 = 长期经营资产净投资额 + 净合并额

其中：
长期经营资产净投资额 = 购建固定资产、无形资产及其他长期资产所支付的现金 
                     - 处置固定资产、无形资产及其他长期资产收到的现金

净合并额 = 取得子公司、合营联营企业及其他营业单位支付的现金净额 
         - 处置子公司、合营联营企业及其他营业单位收到的现金净额
```

#### 扩张性资本支出
```
扩张性资本支出 = 长期经营资产扩张性资本支出 + 净合并额

其中：
长期经营资产扩张性资本支出 = 长期经营资产净投资额 - 折旧及摊销合计

折旧及摊销合计 = 固定资产折旧、油气资产折耗、生产性生物资产折旧 
               + 无形资产摊销 
               + 长期待摊费用摊销 
               + 处置固定资产、无形资产和其他长期资产的损失 
               + 固定资产报废损失
```

#### 扩张性资本支出占长期资产期初净额的比例
```
扩张性资本支出占长期资产期初净额的比例 = 扩张性资本支出 / 期初长期经营资产合计 × 100%
```

**注意**: 
- 所有这些指标都是在现金流量表重构时已经计算好的
- HTML生成时直接从重构后的现金流量表中读取

**数据来源**: 资产负债表（长期经营资产）+ 现金流量表（资本支出、折旧摊销）  
**单位**: 亿元（柱状图）、百分比（折线图）  
**说明**: 展示长期资产投资情况

## 8. 投资收益分析图表（1个）

### 8.1 图表1：投资收益

**图表ID**: `chart_investment_income`  
**图表类型**: 堆叠柱状图+折线图

**数据项**:
- **柱状图**:
  - 长期股权投资收益
  - 长期股权外投资收益（计算字段）
- **折线图**:
  - 长期股权投资收益率（计算字段）

**计算公式**:

#### 长期股权外投资收益
```
长期股权外投资收益 = 投资收益 - 长期股权投资收益
```

#### 长期股权投资收益率
```
长期股权投资收益率 = 长期股权投资收益 / 长期股权投资 × 100%

注意：
- 不使用平均值，直接使用当期长期股权投资
- 长期股权投资收益从利润表（重构后）获取
- 长期股权投资从资产负债表（重构后）获取
```

**数据来源**: 资产负债表（长期股权投资）+ 利润表（长期股权投资收益、投资收益）  
**单位**: 亿元（柱状图）、百分比（折线图）  
**说明**: 展示长期股权投资的盈利能力分析

## 9. 自由现金流分析图表（2个）

### 9.1 图表1：根据EBIT计算的FCFF

**图表ID**: `chart_fcff_ebit`  
**图表类型**: 柱状图

**数据项**:
- 息税前经营利润（计算字段，实际为税后值）
- 折旧及摊销合计（计算字段）
- 资本支出总额
- 营运资本变化量（计算字段）
- FCFF（计算字段）

**计算公式**:

#### 息税前经营利润（税后）
```
息税前经营利润税后 = 息税前经营利润 × (1 - 实际所得税税率)
```

#### 折旧及摊销合计
```
折旧及摊销合计 = 固定资产折旧、油气资产折耗、生产性生物资产折旧 
               + 无形资产摊销 
               + 长期待摊费用摊销 
               + 处置固定资产、无形资产和其他长期资产的损失 
               + 固定资产报废损失
```

#### 营运资本变化量
```
营运资本变化量 = 当期周转性经营投入合计 - 上期周转性经营投入合计

注意：
- 日期列是倒序的（最新在左），所以上期是 i+1
- 第一期（最早的年份）没有上期数据，营运资本变化量为 None
```

#### FCFF
```
FCFF = 息税前经营利润税后 + 折旧及摊销合计 - 资本支出总额 - 营运资本变化量
```

**数据来源**: 利润表（息税前经营利润、实际所得税税率）+ 现金流量表（折旧摊销、资本支出总额）+ 资产负债表（周转性经营投入合计）  
**单位**: 亿元  
**说明**: 基于EBIT计算的企业自由现金流（FCFF），展示各组成部分

---

### 9.2 图表2：FCFE vs Dividend

**图表ID**: `chart_fcfe_dividend`  
**图表类型**: 柱状图

**数据项**:
- FCFE（计算字段）
- 现金分红

**计算公式**:

#### FCFE
```
FCFE = FCFF - 税后真实财务费用 - 净债务增加

其中：
税后真实财务费用 = 真实财务费用 × (1 - 实际所得税税率)
净债务增加 = 当期有息债务合计 - 上期有息债务合计
```

**数据来源**: 利润表（真实财务费用、实际所得税税率）+ 资产负债表（有息债务）+ FCFF计算结果  
**单位**: 亿元  
**说明**: 股权自由现金流与实际分红对比

## 10. TTM数据特殊处理

### 10.1 比率类指标重新计算

在 `annual_report_generator.py` 中，TTM的比率类指标会被重新计算，不能使用加减法公式：

#### 利润表比率
```python
def _recalculate_income_ratios(df, ttm_col):
    营业收入 = get_value('营业收入')
    净利润 = get_value('净利润')
    
    # 重新计算比率
    净利润率 = 净利润 / 营业收入
    毛利率 = 毛利 / 营业收入
    ...
```

#### 现金流量表比率
```python
def _recalculate_cashflow_ratios(df, ttm_col):
    经营现金流 = get_value('经营活动产生的现金流量净额')
    净利润 = get_value('净利润')
    
    # 重新计算比率
    净利润现金含量 = 经营现金流 / 净利润
    ...
```

### 10.2 为什么不能用加减法

**错误示例**（TTM净利润率）:
```
TTM净利润率 = 当前Q3净利润率 - 去年Q3净利润率 + 去年Q4净利润率  ❌
```

**正确方法**:
```
TTM净利润 = 当前Q3净利润 - 去年Q3净利润 + 去年Q4净利润  ✓
TTM营业收入 = 当前Q3营业收入 - 去年Q3营业收入 + 去年Q4营业收入  ✓
TTM净利润率 = TTM净利润 / TTM营业收入  ✓
```

## 13. 常见问题

### 13.1 比率计算错误

**问题**: TTM比率不等于分子/分母

**原因**: 使用了加减法公式计算比率

**解决**: 在 `annual_report_generator.py` 中重新计算比率

### 13.2 总费用率计算错误

**问题**: 总费用率与Excel不一致

**原因**: 
1. 多加了研发费用率
2. 营业外收支净额率应该是减去而不是加上

**解决**: 修正 `_add_operating_expense_calculations()` 中的公式

### 13.3 ROIC计算错误

**问题**: ROIC值异常

**原因**: 
1. 未使用平均投入资本
2. 第一期数据处理不当

**解决**: 正确计算平均值，第一期使用期末值
