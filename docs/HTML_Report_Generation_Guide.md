# HTML财务分析报告生成技术文档

## 1. 概述

### 1.1 文档目的
本文档详细说明 `financial_report.html` 的生成逻辑、图表配置、指标计算公式和数据来源。

### 1.2 核心文件
- **生成器**: `html_report_generator.py`
- **模板**: `templates/financial_report_template.html`
- **输出**: `data/{stock_code}_financial_report.html`

### 1.3 报告结构
HTML报告包含以下主要部分：
1. **资产负债表分析** - 6个图表
2. **利润表分析** - 4个图表
3. **现金流量表分析** - 3个图表
4. **运营效率分析** - 3个图表

---

## 2. 数据源说明

### 2.1 输入数据
报告生成需要以下CSV文件：
- `{stock_code}_balance_sheet_annual_ttm.csv` - 资产负债表年报+TTM
- `{stock_code}_income_statement_annual_ttm.csv` - 利润表年报+TTM
- `{stock_code}_cashflow_statement_annual_ttm.csv` - 现金流量表年报+TTM

### 2.2 数据格式
所有CSV文件格式：
```
项目,2025Q3-TTM,2024/12/31,2023/12/31,...
营业收入,15421000000,15127000000,12168575000,...
净利润,1403000000,1515000000,1216857500,...
...
```

### 2.3 日期列识别
- 年报列格式：`YYYY/12/31` (如 `2024/12/31`)
- TTM列格式：`YYYYQN-TTM` (如 `2025Q3-TTM`)
- 按时间倒序排列（最新在左）

## 3. 资产负债表图表

### 3.1 图表1：总资产与净资产

**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - 总资产 (直接读取)
  - 净资产 (直接读取，即所有者权益合计)
- **折线图**:
  - 有息债务率 (计算字段)

**计算公式**:

#### 有息债务率
```
有息债务率 = 有息债务 / 总资产 × 100%

其中：
有息债务 = 短期借款 + 一年内到期的非流动负债 + 长期借款 + 应付债券 + 长期应付款
```

**数据来源**: 资产负债表
**单位**: 亿元（金额）、百分比（比率）

### 4.2 图表2：息税前利润分析

**图表类型**: 柱状图+折线图

**数据项**:

息税前经营利润率 = 息税前经营利润 / 营业收入 × 100%

### 4.3 图表3：营业费用

**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - 销售费用 (直接读取)
  - 管理费用 (直接读取)
  - 研发费用 (直接读取)
  - 税金及附加 (直接读取)
  - 资产减值损失 (直接读取)
  - 信用减值损失 (直接读取)
  - 营业外收入 (直接读取)
  - 营业外支出 (直接读取)
- **折线图**:
  - 销售费用率 (直接读取)
  - 管理费用率 (直接读取)
  - 研发费用率 (直接读取)
  - 税金及附加率 (计算字段)
  - 经营资产减值损失率 (计算字段)
  - 营业外收支净额率 (计算字段)
  - 总费用率 (计算字段)

**计算公式**:

#### 税金及附加率
```
税金及附加率 = 税金及附加 / 营业收入 × 100%
```

#### 经营资产减值损失率
```
经营资产减值损失率 = (资产减值损失 + 信用减值损失) / 营业收入 × 100%
```

#### 营业外收支净额
```
营业外收支净额 = 营业外收入 - 营业外支出
```

#### 营业外收支净额率
```
营业外收支净额率 = 营业外收支净额 / 营业收入 × 100%
```

#### 总费用率
```
总费用率 = 税金及附加率 + 销售费用率 + 管理费用率 + 经营资产减值损失率 - 营业外收支净额率
```

**注意**: 
- 总费用率中**不包含**研发费用率
- 营业外收支净额率是**减去**的，不是加上

**数据来源**: 利润表
**单位**: 亿元（金额）、百分比（比率）

### 4.4 图表4：股权价值增加值

**图表类型**: 柱状图

**数据项**:
- 净利润 (直接读取)
- 股权资本成本（默认8%） (直接读取)
- 股权价值增加值 (直接读取)

**计算公式**:

#### 股权资本成本
```
股权资本成本 = 所有者权益合计 × 股权资本成本率

其中：
股权资本成本率 = 8%（默认值，可配置）
```

#### 股权价值增加值
```
股权价值增加值 = 净利润 - 股权资本成本
```

**数据来源**: 利润表（净利润）+ 资产负债表（所有者权益）
**单位**: 亿元

## 5. 现金流量表图表

### 5.1 图表1：自由现金流

**图表类型**: 柱状图+折线图

**数据项**:
- **柱状图**:
  - 企业自由现金流FCFF (直接读取)
  - 股权自由现金流FCFE (直接读取)
- **折线图**:
  - 自由现金流/营业收入 (计算字段)

**计算公式**:

#### 企业自由现金流FCFF
```
FCFF = 息前税后经营利润 + 折旧摊销 - 经营营运资本增加 - 资本支出
```

#### 股权自由现金流FCFE
```
FCFE = FCFF - 税后真实财务费用 - 净债务增加
```

#### 自由现金流/营业收入
```
自由现金流/营业收入 = FCFF / 营业收入 × 100%
```

**数据来源**: 现金流量表
**单位**: 亿元（金额）、百分比（比率）

## 6. 运营效率分析图表

### 6.1 图表1：ROIC分析

**图表类型**: 折线图

**数据项**:
- ROIC (计算字段)

**计算公式**:

#### ROIC（投入资本回报率）
```
ROIC = 息前税后经营利润 / 平均投入资本 × 100%

其中：
平均投入资本 = (期初投入资本 + 期末投入资本) / 2
投入资本 = 经营营运资本 + 经营长期资本
```

**特殊处理**:
- 第一期数据：使用期末投入资本（无期初数据）
- 其他期：使用平均投入资本

**数据来源**: 利润表（息前税后经营利润）+ 资产负债表（投入资本）
**单位**: 百分比

---

### 6.2 图表2：运营效率指标

**图表类型**: 折线图

**数据项**:
- 总资产周转率 (计算字段)
- 应收账款周转率 (计算字段)
- 存货周转率 (计算字段)

**计算公式**:

#### 总资产周转率
```
总资产周转率 = 营业收入 / 平均总资产

其中：
平均总资产 = (期初总资产 + 期末总资产) / 2
```

#### 应收账款周转率
```
应收账款周转率 = 营业收入 / 平均应收账款及票据

其中：
平均应收账款及票据 = (期初应收账款及票据 + 期末应收账款及票据) / 2
```

#### 存货周转率
```
存货周转率 = 营业成本 / 平均存货

其中：
平均存货 = (期初存货 + 期末存货) / 2
```

**特殊处理**:
- 第一期数据：使用期末值（无期初数据）
- 其他期：使用平均值

**数据来源**: 利润表（营业收入、营业成本）+ 资产负债表（资产项）
**单位**: 次数

---

### 6.3 图表3：杜邦分析

**图表类型**: 折线图

**数据项**:
- ROE (计算字段)
- 净利润率 (直接读取)
- 总资产周转率 (计算字段)
- 权益乘数 (计算字段)

**计算公式**:

#### ROE（净资产收益率）
```
ROE = 净利润 / 平均净资产 × 100%

其中：
平均净资产 = (期初净资产 + 期末净资产) / 2
```

#### 杜邦分析公式
```
ROE = 净利润率 × 总资产周转率 × 权益乘数

其中：
净利润率 = 净利润 / 营业收入
总资产周转率 = 营业收入 / 平均总资产
权益乘数 = 平均总资产 / 平均净资产
```

**特殊处理**:
- 第一期数据：使用期末值（无期初数据）
- 其他期：使用平均值

**数据来源**: 利润表（净利润、营业收入）+ 资产负债表（总资产、净资产）
**单位**: 百分比（ROE、净利润率）、次数（周转率、乘数）

## 8. TTM数据特殊处理

### 8.1 比率类指标重新计算

在 `annual_report_generator.py` 中，TTM的比率类指标会被重新计算，不能使用加减法公式：

#### 利润表比率
```python
def _recalculate_income_ratios(df, ttm_col):
    营业收入 = get_value('营业收入')
    净利润 = get_value('净利润')
    
    # 重新计算比率
    净利润率 = 净利润 / 营业收入
    ...
```

#### 现金流量表比率
```python
def _recalculate_cashflow_ratios(df, ttm_col):
    经营现金流 = get_value('经营活动产生的现金流量净额')
    净利润 = get_value('净利润')
    
    # 重新计算比率
    净利润现金含量 = 经营现金流 / 净利润
    ...
```

### 8.2 为什么不能用加减法

**错误示例**（TTM净利润率）:
```
TTM净利润率 = 当前Q3净利润率 - 去年Q3净利润率 + 去年Q4净利润率  ❌
```

**正确方法**:
```
TTM净利润 = 当前Q3净利润 - 去年Q3净利润 + 去年Q4净利润  ✓
TTM营业收入 = 当前Q3营业收入 - 去年Q3营业收入 + 去年Q4营业收入  ✓
TTM净利润率 = TTM净利润 / TTM营业收入  ✓
```

## 9. 数据单位处理

### 9.1 金额单位转换

所有金额数据在显示时需要转换为亿元：

```python
value_in_yi = value / 100000000  # 原始单位：元
```

### 9.2 比率单位转换

所有比率数据在显示时需要转换为百分比：

```python
percentage = ratio * 100  # 原始单位：小数
```

### 9.3 周转率单位

周转率保持原值（次数），不需要转换。

## 11. 常见问题

### 11.1 比率计算错误

**问题**: TTM比率不等于分子/分母

**原因**: 使用了加减法公式计算比率

**解决**: 在 `annual_report_generator.py` 中重新计算比率

### 11.2 总费用率计算错误

**问题**: 总费用率与Excel不一致

**原因**: 
1. 多加了研发费用率
2. 营业外收支净额率应该是减去而不是加上

**解决**: 修正 `_add_operating_expense_calculations()` 中的公式

### 11.3 ROIC计算错误

**问题**: ROIC值异常

**原因**: 
1. 未使用平均投入资本
2. 第一期数据处理不当

**解决**: 正确计算平均值，第一期使用期末值
