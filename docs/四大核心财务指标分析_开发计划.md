# 四大核心财务指标分析 - 开发计划

## 项目概述

### 目标
实现四大核心财务指标分析，包含全A股对比功能，用于评估上市公司的财务质量和竞争力。

### 核心策略
**指标计算与展示方案**：
- **计算范围**：为每个公司计算其所有历史节点的四大指标（从上市至今）
- **HTML报告展示**：
  - 默认显示最近10年数据
  - 用户可通过拖动时间轴滑块查看更长时间段的历史数据
  - 图表间的时间轴保持同步缩放
  - 完全复用当前项目的 `html_report_generator.py` 实现机制
- **优势**：
  - 保留完整历史趋势，便于长期分析
  - 自动适配不同公司的上市时间（有的20年，有的5年）
  - 用户体验与现有报告一致

### 数据范围
- **股票范围**：全A股（排除北交所，包含ST股票）约5000+只
- **时间跨度**：所有历史数据（从上市至今的完整历史）
- **数据源**：4张财务报表（资产负债表、利润表、现金流量表、财务指标）
- **暂不获取**：分红数据、总股本数据（减少API调用）
- **报告展示**：HTML报告默认显示最近10年，用户可拖动时间轴查看完整历史

### API调用量估算
- **每只股票**：4张报表 ≈ 4-5次API调用
- **全A股5000只**：5000 × 5 = 2.5万次API调用
- **账户限额**：2000积分，每天2万次
- **预计完成时间**：1.5-2天

### 关键实现说明

**1. 数据获取策略**
- **获取全部历史数据**：每只股票获取其完整的历史财务数据，相当于把Tushare上全A股的所有财务数据都下载到本地数据库
- **数据完整性**：参考当前项目的实现方式（`main.py`中的`get_all_financial_data`方法），确保获取完整的字段
- **数据存储**：所有数据存入SQLite数据库，方便后续使用和扩展

**2. TTM计算规则**
- **参考现有实现**：使用`annual_report_generator.py`中的TTM计算逻辑
- **年报优先**：对于已有完整年报的年份（如2024、2023），直接使用年报数据，不计算该年的TTM
- **TTM适用场景**：只在最新数据不是Q4时计算TTM（如2025Q3-TTM）
- **计算公式**：TTM = 今年Q累计 - 去年同Q累计 + 去年Q4累计

**3. 多线程并发处理**
- **数据库写入安全**：使用连接池和事务机制，避免多线程写入时的死锁问题
- **写入策略**：每个线程使用独立的数据库连接，写入时使用行级锁
- **批量提交**：积累一定数量的数据后批量提交，提高写入效率

## 四大指标定义

### 指标1：报表逻辑一致性检验暨回款周期

**子指标1：应收账款周转率对数**
```
应收账款周转率对数 = ln(TTM营业收入 / 平均应收账款)

其中：
- TTM营业收入 = 最近四个季度的单季度营业收入之和
- 平均应收账款 = (期初应收账款 + 期末应收账款) / 2
```

**子指标2：毛利率**
```
毛利率 = (营业收入 - 营业成本) / 营业收入
```

**分析目的**：
- 通过应收账款周转率和毛利率的联合分析，检验报表逻辑一致性
- 如果周转率下降但毛利率上升，可能存在虚增收入风险

### 指标2：再投资质量暨跑冒滴漏风险检验

**长期经营资产周转率对数**
```
长期经营资产周转率对数 = ln(TTM营业收入 / 平均长期经营资产)

其中：
- 平均长期经营资产 = (期初长期经营资产 + 期末长期经营资产) / 2
- 长期经营资产 = 固定资产 + 在建工程 + 生产性生物资产 + 公益性生物资产 
                + 油气资产 + 使用权资产 + 无形资产 + 开发支出 
                + 商誉 + 长期待摊费用 + 其他非流动资产
```

**分析目的**：
- 评估长期资产使用效率和再投资质量
- 周转率下降可能意味着资产利用效率下降或存在跑冒滴漏

### 指标3：产业链地位检验暨资金运用能力

**营运净资本比率**
```
营运净资本比率 = 营运净资本 / 总资产 × 100%

其中：
- 营运净资本 = 应收账款 + 应收票据 + 应收款项融资 + 合同资产 
              - 应付账款 - 应付票据 - 合同负债
```

**解读**：
- **负值**：公司占用上下游资金（产业链地位强）
- **正值**：被上下游占用资金（产业链地位弱）

### 指标4：真实盈利水平暨现金流创造能力

**经营现金流比率**
```
经营现金流比率 = 经营活动产生的现金流量净额 / 总资产 × 100%
```

**分析目的**：
- 评估真实盈利能力和现金创造能力
- 数值越高说明盈利质量越好

**多线程安全的数据库写入**：
```python
class MarketDataManager:
    def __init__(self, db_path):
        self.db_path = db_path
        # 使用线程本地存储，每个线程独立连接
        self.local = threading.local()
    
    def get_connection(self):
        """获取线程本地的数据库连接"""
        if not hasattr(self.local, 'conn'):
            self.local.conn = sqlite3.connect(
                self.db_path,
                timeout=30.0,  # 30秒超时
                isolation_level='DEFERRED'  # 延迟锁定
            )
            # 启用WAL模式，支持并发读写
            self.local.conn.execute('PRAGMA journal_mode=WAL')
        return self.local.conn
    
    def save_financial_data(self, ts_code, end_date, data_type, data):
        """
        保存财务数据（线程安全）
        使用 INSERT OR REPLACE 避免重复
        """
        conn = self.get_connection()
        try:
            conn.execute('BEGIN IMMEDIATE')  # 立即获取写锁
            # 执行插入操作
            # ...
            conn.commit()
        except sqlite3.OperationalError as e:
            if 'locked' in str(e):
                # 数据库锁定，等待后重试
                time.sleep(0.1)
                return self.save_financial_data(ts_code, end_date, data_type, data)
            else:
                conn.rollback()
                raise
```

**TTM计算（参考 annual_report_generator.py）**：
```python
def calculate_ttm_revenue(self, ts_code, end_date):
    """
    计算TTM营业收入
    参考 annual_report_generator.py 中的 _calculate_ttm 方法
    
    重要规则：
    1. 如果最新数据是Q4（12月31日），直接使用年报数据，不计算TTM
    2. 只在最新数据是Q1-Q3时才计算TTM
    3. TTM公式：今年Q累计 - 去年同Q累计 + 去年Q4累计
    """
    # 判断是否为Q4
    quarter = self._get_quarter(end_date)
    
    if quarter == 4:
        # Q4直接使用年报数据
        return self._get_annual_revenue(ts_code, end_date)
    
    # Q1-Q3计算TTM
    latest_year = int(end_date[:4])
    last_year = latest_year - 1
    
    # 今年Q累计
    current_q_value = self._get_revenue_cumulative(ts_code, end_date)
    
    # 去年同Q累计
    last_year_same_q_date = f"{last_year}{end_date[4:]}"
    last_year_same_q_value = self._get_revenue_cumulative(ts_code, last_year_same_q_date)
    
    # 去年Q4累计（即去年年报）
    last_year_q4_date = f"{last_year}1231"
    last_year_q4_value = self._get_revenue_cumulative(ts_code, last_year_q4_date)
    
    # 计算TTM
    if pd.isna(current_q_value) or pd.isna(last_year_q4_value):
        return np.nan
    
    # 如果去年同Q数据为空（说明是Q1），设为0
    if pd.isna(last_year_same_q_value):
        last_year_same_q_value = 0
    
    ttm = current_q_value - last_year_same_q_value + last_year_q4_value
    return ttm
```

## 技术要点

### 1. TTM计算

```python
def calculate_ttm_revenue(ts_code, end_date):
    """
    计算TTM营业收入
    
    逻辑：
    1. 获取最近4个季度数据
    2. 如果是累计值，转换为单季度值
    3. 求和得到TTM
    """
    # 获取最近4个季度
    quarters = get_recent_quarters(end_date, n=4)
    
    ttm_revenue = 0
    for i, quarter in enumerate(quarters):
        revenue_cumulative = get_revenue(ts_code, quarter)
        
        # 如果是Q1，直接使用
        if quarter.endswith('0331'):
            revenue_single = revenue_cumulative
        else:
            # 否则，减去上一季度累计值
            prev_quarter = get_previous_quarter(quarter)
            revenue_prev = get_revenue(ts_code, prev_quarter)
            revenue_single = revenue_cumulative - revenue_prev
        
        ttm_revenue += revenue_single
    
    return ttm_revenue
```

### 2. 平均值计算

```python
def calculate_average(ts_code, field_name, end_date):
    """
    计算平均值（期初 + 期末）/ 2
    
    期初 = 上一期的期末
    """
    # 期末值
    value_end = get_field_value(ts_code, field_name, end_date)
    
    # 期初值 = 上一期期末
    prev_date = get_previous_period(end_date)
    value_begin = get_field_value(ts_code, field_name, prev_date)
    
    # 如果期初值缺失，只用期末值
    if value_begin is None:
        return value_end
    
    return (value_begin + value_end) / 2
```

### 3. 多线程 + 限流

```python
import threading
import time
from queue import Queue

class RateLimiter:
    """API限流器：200次/分钟"""
    def __init__(self, max_calls=200, period=60):
        self.max_calls = max_calls
        self.period = period
        self.calls = []
        self.lock = threading.Lock()
    
    def wait(self):
        with self.lock:
            now = time.time()
            # 移除过期的调用记录
            self.calls = [t for t in self.calls if now - t < self.period]
            
            # 如果达到限制，等待
            if len(self.calls) >= self.max_calls:
                sleep_time = self.period - (now - self.calls[0])
                time.sleep(sleep_time)
                self.calls = []
            
            # 记录本次调用
            self.calls.append(time.time())
```

- **应对**：缓存机制 + 增量计算
