# 年报+TTM重构报表生成器使用指南

## 功能概述

生成年报和TTM（滚动12个月）数据组成的重构报表，包含过去10年年报数据和最新一期TTM数据。

## TTM计算规则

### 什么是TTM？

TTM (Trailing Twelve Months) = 滚动12个月数据，用于消除季节性波动影响。

### 适用范围

| 报表类型 | 数据性质 | TTM计算 |
|---------|---------|---------|
| 资产负债表 | 时点数据 | **不需要**，直接取最新季度数据 |
| 利润表 | 期间数据（累计值） | **需要**，计算滚动12个月 |
| 现金流量表 | 期间数据（累计值） | **需要**，计算滚动12个月 |

### 计算公式

**数据库数据特点**：季度数据为累计值
```
Q1 = 1-3月累计
Q2 = 1-6月累计
Q3 = 1-9月累计
Q4 = 1-12月累计（年度数据）
```

**TTM计算公式**（以Q3为例）：
```
TTM = 今年Q3累计 - 去年Q3累计 + 去年Q4累计
    = 今年Q3 + 去年Q4单季度
    = 最近12个月滚动数据
```

**示例**（2025Q3-TTM）：
```
TTM = 2025Q3累计 - 2024Q3累计 + 2024Q4累计
    = (2025Q1+Q2+Q3) - (2024Q1+Q2+Q3) + (2024Q1+Q2+Q3+Q4)
    = 2024Q4 + 2025Q1+Q2+Q3
    = 最近12个月
```

**特殊情况**：
- 如果最新报告期是Q4（年报），则直接使用年报数据，不需要计算TTM

## 使用方法

### 方法1：命令行方式（推荐）

```bash
# 1. 先获取重构数据
python main.py 603345.SH --start-date 20150101

# 2. 生成年报+TTM报表
python main.py 603345.SH --annual-ttm

# 或者一步完成（推荐）
python main.py 603345.SH --start-date 20150101 --annual-ttm

# 指定年数（默认10年）
python main.py 603345.SH --annual-ttm --years 15

# 输出Excel格式
python main.py 603345.SH --annual-ttm --format excel
```

### 方法2：使用示例脚本

```bash
# 修改 example_annual_ttm.py 中的股票代码
python example_annual_ttm.py
```

### 方法3：编程调用

```python
from annual_report_generator import AnnualReportGenerator
import pandas as pd

# 读取重构后的数据
df_balance = pd.read_csv('data/603345.SH_balancesheet_restructured.csv')
df_income = pd.read_csv('data/603345.SH_income_restructured.csv')
df_cashflow = pd.read_csv('data/603345.SH_cashflow_restructured.csv')

# 生成年报+TTM数据
generator = AnnualReportGenerator()
reports = generator.generate_annual_reports_with_ttm(
    df_balance, df_income, df_cashflow, years=10
)

# 格式化并保存
for report_name, df_report in reports.items():
    df_formatted = generator.format_annual_report(df_report, report_name)
    df_formatted.to_csv(f'{report_name}_annual_ttm.csv', index=False, encoding='utf-8-sig')
```

## 应用场景

### 1. 长期趋势分析
追踪企业10年财务数据变化趋势，了解企业发展轨迹。

### 2. TTM数据对比
消除季节性波动影响，便于进行同比分析。

### 3. 投资决策支持
提供最新12个月滚动数据，更准确反映企业当前经营状况。

### 4. 报表整合输出
一次性生成三大报表的年报+TTM数据，便于整体分析。

## 注意事项

1. **数据要求**：需要先运行 `main.py` 生成重构后的财务数据
2. **日期范围**：建议获取至少10年的数据以确保年报数据完整
3. **Q4年报**：如果最新报告期是Q4，则TTM列就是年报数据
4. **数据完整性**：TTM计算需要去年同期数据，如果缺少相关数据会导致TTM值为空

## 常见问题

### Q1: 为什么资产负债表不计算TTM？
A: 资产负债表是时点数据（某一天的快照），不是累计值，所以直接使用最新季度数据即可。

### Q2: TTM值为什么会为空？
A: TTM计算需要去年同期数据和去年Q4数据，如果缺少这些数据，TTM值会为空。建议获取足够长的历史数据。

### Q3: 如何判断是否需要计算TTM？
A: 程序会自动判断最新报告期季度。如果是Q4，则不需要计算TTM；如果是Q1-Q3，则自动计算TTM。

### Q4: 可以指定不同的年数吗？
A: 可以，使用 `--years` 参数指定年数，例如 `--years 15`。
