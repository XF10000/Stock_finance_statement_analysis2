## 关键发现和修正

### 1. 金融资产字段遗漏

**问题**：金融资产字段列表中缺少"可供出售金融资产"、"持有至到期投资"、"应收款项类投资"

**修正**：在`balance_sheet_restructure.py`中添加这些字段

**效果**：
- 2018年金融资产合计差异：从71.70%降到5.28%
- 平均差异率：从20.81%降到4.51%

### 2. "其他非流动资产"分类错误

**问题**：将"其他非流动资产"错误归类为长期经营资产

**修正**：移到金融资产字段列表

**效果**：经营资产合计差异率从最高223%降到0.71%以内

### 3. 其他综合收益处理不当

**问题**：未将"其他综合收益"计入金融资产收益

**发现**：参考文件将"其他综合收益的税后净额"直接计入息税前金融资产收益

**修正**：将"其他综合收益的税后净额"还原成税前金额，计入息税前金融资产收益

**效果**：
- 息前税后金融资产收益差异：从56.96%降到12.60%
- 息前税后利润总额差异：从9.59%降到8.65%

## 仍存在的问题

### 1. 税前利润差异（20.72%）

**原因**：息税前利润总额计算存在差异

**差异链条**：
```
税前利润差异 (20.72%)
  ← 息税前利润总额差异 (16.94%)
    ← 息前税后利润总额差异 (8.65%)
      ← 息前税后金融资产收益差异 (12.60%)
```

**分析**：
- 我将其他综合收益还原成税前金额（使用实际税率）
- 参考文件可能使用了不同的税率或处理方式
- 2022年差异示例：我的计算30.41亿 vs 参考26.19亿

### 2. 金融资产合计差异（2016年16.50%）

**可能原因**：
- 早期年报中可能包含其他金融资产项目
- 部分字段在年报格式变更前后的处理方式不同
