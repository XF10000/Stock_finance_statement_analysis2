## 3. 重构规则详细说明

### 3.1 资产资本表（资产负债表重构）

#### 3.1.1 重构目标
将传统资产负债表重构为"资产-资本"结构，区分金融资产、经营资产、有息债务、所有者权益。

#### 3.1.2 结构设计

```
资产结构
├── 金融资产合计
│   ├── 货币资金
│   ├── 拆出资金
│   ├── 交易性金融资产
│   ├── 衍生金融资产(流动)
│   ├── 应收利息
│   ├── 应收股利
│   ├── 买入返售金融资产
│   ├── 持有待售资产
│   ├── 一年内到期的非流动资产
│   ├── 发放贷款及垫款(流动)
│   ├── 发放贷款及垫款(非流动)
│   ├── 债权投资
│   ├── 其他债权投资
│   ├── 其他权益工具投资
│   ├── 其他非流动金融资产
│   └── 投资性房地产
├── 长期股权投资
└── 经营资产合计
    ├── 周转性经营投入合计 = 营运资产小计 - 营运负债小计
    │   ├── 营运资产小计
    │   │   ├── 应收票据
    │   │   ├── 应收账款
    │   │   ├── 应收款项融资
    │   │   ├── 预付款项
    │   │   ├── 其他应收款（扣除应收利息、应收股利）
    │   │   ├── 存货
    │   │   ├── 合同资产
    │   │   ├── 长期应收款
    │   │   └── 其他流动资产
    │   └── 营运负债小计
    │       ├── 应付票据
    │       ├── 应付账款
    │       ├── 预收账款
    │       ├── 合同负债
    │       ├── 应付职工薪酬
    │       ├── 长期应付职工薪酬
    │       ├── 应交税费
    │       ├── 其他应付款（扣除应付利息、应付股利）
    │       └── 其他流动负债
    └── 长期经营资产合计
        ├── 固定资产
        ├── 在建工程
        ├── 生产性生物资产
        ├── 公益性生物资产
        ├── 油气资产
        ├── 使用权资产
        ├── 无形资产
        ├── 开发支出
        ├── 商誉
        ├── 长期待摊费用
        ├── 其他非流动资产
        ├── 递延所得税资产
        └── 递延所得税负债

资本结构
├── 有息债务合计 = 短期债务 + 长期债务
│   ├── 短期债务
│   │   ├── 短期借款
│   │   ├── 向中央银行借款
│   │   ├── 拆入资金
│   │   ├── 交易性金融负债
│   │   ├── 衍生金融负债
│   │   ├── 卖出回购金融资产
│   │   ├── 吸收存款及同业存放
│   │   ├── 代理买卖证券款
│   │   ├── 代理承销证券款
│   │   ├── (其中)应付利息
│   │   ├── 持有待售负债
│   │   └── 一年内到期的非流动负债
│   └── 长期债务
│       ├── 长期借款
│       ├── 应付债券
│       ├── 租赁负债
│       └── 长期应付款
├── 所有者权益合计
│   ├── 归属于母公司股东权益合计
│   │   ├── 股本
│   │   ├── 其他权益工具
│   │   ├── (其中)优先股
│   │   ├── (其中)永续债
│   │   ├── 资本公积
│   │   ├── 减：库存股
│   │   ├── 其他综合收益
│   │   ├── 专项储备
│   │   ├── 盈余公积
│   │   ├── 一般风险准备金
│   │   └── 未分配利润
│   └── 少数股东权益
└── 资本总额 = 所有者权益 + 有息债务
```

#### 3.1.3 核心计算公式

| 项目 | 计算规则 |
|------|---------|
| 其他应收款 | = 原始其他应收款 - 应收利息 - 应收股利（需手动计算） |
| 其他应付款 | = 原始其他应付款 - 应付利息 - 应付股利（需手动计算） |
| 周转性经营投入合计 | = 营运资产小计 - 营运负债小计 |
| 经营资产合计 | = 周转性经营投入合计 + 长期经营资产合计 |
| 资产总额 | = 金融资产合计 + 长期股权投资 + 经营资产合计 |
| 有息债务合计 | = 短期债务 + 长期债务 |
| 所有者权益合计 | = 归属母公司股东权益 + 少数股东权益 |
| 资本总额 | = 所有者权益合计 + 有息债务合计 |

> **注意**：
> - API获取的数据中的“其他应收款”字段未扣除应收利息和应收股利，需手动计算
> - 递延所得税资产和递延所得税负债全部归入经营资产，不做经营性/金融性区分

#### 3.1.4 小计项目详细计算公式

**金融资产合计**：
```
= 货币资金 + 拆出资金 + 交易性金融资产 + 衍生金融资产(流动) + 应收利息 + 应收股利 
  + 买入返售金融资产 + 持有待售资产 + 一年内到期的非流动资产 + 发放贷款及垫款(流动) 
  + 发放贷款及垫款(非流动) + 债权投资 + 其他债权投资 + 其他权益工具投资 
  + 其他非流动金融资产 + 投资性房地产
```

**营运资产小计**：
```
= 应收票据 + 应收账款 + 应收款项融资 + 预付款项 + 其他应收款 + 存货 + 合同资产 
  + 长期应收款 + 其他流动资产 + 结算备付金 + 应收保费 + 应收分保账款 
  + 应收分保合同准备金 + 保险合同准备金
```

**营运负债小计**：
```
= 应付票据 + 应付账款 + 预收账款 + 合同负债 + 应付职工薪酬 + 长期应付职工薪酬 
  + 应交税费 + 其他应付款 + 应付手续费及佣金 + 应付分保账款 + 其他流动负债 
  + 预计负债(流动) + 预计负债(非流动) + 一年内到期的递延收益 + 长期递延收益 
  + 其他非流动负债
```

**长期经营资产合计**：
```
= 固定资产 + 在建工程 + 生产性生物资产 + 公益性生物资产 + 油气资产 + 使用权资产 
  + 无形资产 + 开发支出 + 商誉 + 长期待摊费用 + 其他非流动资产 + 递延所得税资产 
  - 递延所得税负债
```
> 注意：递延所得税负债作为减项，因为它本质上是负债而非资产

**短期债务**：
```
= 短期借款 + 向中央银行借款 + 拆入资金 + 交易性金融负债 + 衍生金融负债 
  + 卖出回购金融资产 + 吸收存款及同业存放 + 代理买卖证券款 + 代理承销证券款 
  + 应付利息 + 持有待售负债 + 一年内到期的非流动负债
```

**长期债务**：
```
= 长期借款 + 应付债券 + 租赁负债 + 长期应付款
```

**归属于母公司股东权益合计**：
```
= 股本 + 其他权益工具 + 资本公积 - 库存股 + 其他综合收益 + 专项储备 + 盈余公积 
  + 一般风险准备金 + 未分配利润
```

### 3.2 股权价值增加表（利润表重构）

#### 3.2.1 重构目标
将传统利润表重构为股权价值增加表，计算息税前经营利润、息前税后利润、股权价值增加值等核心指标。

#### 3.2.2 结构设计

```
股权价值增加表
├── 营业收入
├── 营业成本
├── 营业成本率 = 营业成本 / 营业收入
├── 毛利 = 营业收入 - 营业成本
├── 毛利率 = 毛利 / 营业收入
├── 税金及附加
├── 营业税金及附加率 = 税金及附加 / 营业收入
├── 销售费用
├── 销售费用率 = 销售费用 / 营业收入
├── 管理费用
├── 管理费用率 = 管理费用 / 营业收入
├── 研发费用
├── 研发费用率 = 研发费用 / 营业收入
├── 资产减值损失
├── 信用减值损失
├── 资产减值损失率 = (资产减值损失 + 信用减值损失) / 营业收入
├── 资产处置收益
├── 加：其他收益
├── 加：营业外收入
├── 减：营业外支出
├── 营业外收支及其他占营业收入的比例 = （资产处置收益 + 其他收益 + 营业外收入 - 营业外支出）/ 营业收入
├── 息税前经营利润 = 营业收入 - 营业成本 - 税金 - 销售费用 - 管理费用 - 研发费用 - 资产减值损失 + 其他收益 + 营业外收入 - 营业外支出
├── 经营利润所得税
├── 息前税后经营利润 = 息税前经营利润 - 经营利润所得税
├── 投资收益
│   ├── 对联营企业及合营企业的投资收益
│   └── 短期投资收益 = 投资收益 - 对联营企业投资收益
├── 利息收入
├── 净敞口套期收益
├── 公允价值变动收益
├── 汇兑收益
├── 其他综合收益的税后净额
├── 息税前金融资产收益 = 短期投资收益 + 利息收入 + 净敞口套期收益 + 公允价值变动收益 + 汇兑收益 + 其他综合收益
├── 金融资产收益所得税
├── 息前税后金融资产收益 = 息税前金融资产收益 - 金融资产收益所得税
├── 长期股权投资收益
├── 息税前利润总额 = 息税前经营利润 + 息前税后金融资产收益 + 长期股权投资收益
├── 息前税后利润总额
├── 财务费用
├── 真实财务费用 = 财务费用 + 利息收入（加回被扣除的利息收入）
├── (其中)利息费用
├── 财务费用抵税效应
├── 税后真实财务费用 = 真实财务费用 - 财务费用抵税效应
├── 税前利润 = 息税前利润总额 - 真实财务费用
├── 减：所得税费用
├── 实际所得税税率 = 所得税费用 / 税前利润
├── 净利润 = 税前利润 - 所得税费用
├── 净利润率 = 净利润 / 营业收入
├── 股权资本成本（默认8%）
└── 股权价值增加值 = 净利润 - 股权资本成本
```

#### 3.2.3 核心计算公式

| 项目 | 计算规则 |
|------|---------|
| 毛利 | = 营业收入 - 营业成本 |
| 息税前经营利润 | = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 资产减值损失 - 信用减值损失 + 资产处置收益 + 其他收益 + 营业外收入 - 营业外支出 |
| **实际所得税税率** | = 所得税费用 / (税前利润 - 长期股权投资收益) |
| **经营利润所得税** | = 息税前经营利润 × 实际所得税税率 |
| 息前税后经营利润 | = 息税前经营利润 - 经营利润所得税 |
| 短期投资收益 | = 投资收益 - 对联营企业投资收益 |
| 息税前金融资产收益 | = 短期投资收益 + 利息收入 + 净敞口套期收益 + 公允价值变动收益 + 汇兑收益 + 其他综合收益 |
| **金融资产收益所得税** | = 息税前金融资产收益 × 实际所得税税率 |
| 息前税后金融资产收益 | = 息税前金融资产收益 - 金融资产收益所得税 |
| 息税前利润总额 | = 息前税后经营利润 + 息前税后金融资产收益 + 长期股权投资收益 |
| 真实财务费用 | = 财务费用 + 利息收入 |
| **财务费用抵税效应** | = 真实财务费用 × 实际所得税税率 |
| 税后真实财务费用 | = 真实财务费用 - 财务费用抵税效应 |
| 税前利润 | = 息税前利润总额 - 真实财务费用 |
| 净利润 | = 税前利润 - 所得税费用 |
| 股权资本成本 | 默认值：8%（可在配置文件中调整） |
| 股权价值增加值 | = 净利润 - 股权资本成本 |

#### 3.2.4 所得税项目计算说明

**设计原理**：将所得税按利润来源进行分配，分别计算经营利润、金融资产收益、财务费用各自承担的所得税影响。

| 项目 | 公式 | 说明 |
|------|------|------|
| **实际所得税税率** | = 所得税费用 / (税前利润 - 长期股权投资收益) | 排除长期股权投资收益（通常适用不同税率）后的实际税率 |
| **经营利润所得税** | = 息税前经营利润 × 实际所得税税率 | 经营活动产生的利润应承担的所得税 |
| **金融资产收益所得税** | = 息税前金融资产收益 × 实际所得税税率 | 金融资产投资收益应承担的所得税 |
| **财务费用抵税效应** | = 真实财务费用 × 实际所得税税率 | 利息支出可抵税，产生的抵税收益 |

**计算顺序**：
1. 先计算实际所得税税率
2. 分别计算三个所得税相关项目
3. 用所得税项目计算税后利润

**示例**（假设数据）：
```
息税前经营利润 = 100亿
息税前金融资产收益 = 10亿
真实财务费用 = 5亿
所得税费用 = 20亿
长期股权投资收益 = 2亿
税前利润 = 80亿

实际所得税税率 = 20 / (80 - 2) = 25.64%

经营利润所得税 = 100 × 25.64% = 25.64亿
金融资产收益所得税 = 10 × 25.64% = 2.56亿
财务费用抵税效应 = 5 × 25.64% = 1.28亿
```

### 3.3 现金流量分析表

#### 3.3.1 重构目标
分析自由现金流、资本支出、债务筹资等核心现金流指标。

#### 3.3.2 结构设计

```
现金流量分析表
├── 自由现金流量分析
│   ├── 销售商品、提供劳务收到的现金
│   ├── 营业收入
│   ├── 口径一收入现金含量 = 销售收到现金 / 营业收入
│   ├── 购买商品、接收劳务支付的现金
│   ├── 支付给职工及为职工支付的现金
│   ├── 营业总成本
│   ├── 成本费用付现率 = (购买支付现金 / 1.17% + 职工支付现金) / 营业总成本
│   ├── 经营活动产生的现金流量净额
│   ├── 息前税后经营利润
│   ├── 息前税后经营利润现金含量 = 经营现金流 / 息前税后经营利润
│   ├── 净利润
│   └── 净利润现金含量 = 经营现金流 / 净利润
├── 经营活动产生的现金流量净额分析
│   ├── 经营活动产生的现金流量净额
│   ├── 加：资产减值准备
│   ├── 固定资产折旧、油气资产折耗、生产性生物资产折旧
│   ├── 无形资产摊销
│   ├── 长期待摊费用摊销
│   ├── 处置固定资产、无形资产和其他长期资产的损失
│   ├── 固定资产报废损失
│   ├── 非付现成本费用 = 资产减值准备 + 固定资产折旧、油气资产折耗、生产性生物资产折旧 + 无形资产摊销 + 长期待摊费用摊销 + 处置固定资产、无形资产和其他长期资产的损失 + 固定资产报废损失
│   └── 经营资产自由现金流量 = 经营活动现金流净额 - 非付现成本费用
├── 投资活动产生的现金流量分析
│   ├── 收回投资收到的现金
│   ├── 取得投资收益所收到的现金
│   ├── 处置固定资产、无形资产及其他长期资产收到的现金
│   ├── 处置子公司、合营联营企业及其他营业单位收到的现金净额
│   ├── 收到的其他与投资活动相关的现金
│   ├── 投资活动现金流入小计
│   ├── 购建固定资产、无形资产及其他长期资产所支付的现金
│   ├── 投资所支付的现金
```
