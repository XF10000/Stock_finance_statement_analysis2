## 🔧 测试工具

### IntegrationTester 类

**主要方法**:

1. **setup()** - 环境准备
   - 清理旧测试数据
   - 创建测试目录
   - 初始化测试环境

2. **test_phase1_database()** - Phase 1 测试
   - 运行 test_phase1_database.py
   - 验证数据库表结构

3. **test_phase2_methods()** - Phase 2 测试
   - 运行 test_phase2_methods.py
   - 验证数据管理方法

4. **test_end_to_end()** - 端到端测试
   - 使用 Phase 3 测试数据库
   - 运行 main.py 生成报告
   - 验证输出文件

5. **test_edge_cases()** - 边界测试
   - 测试不存在的股票
   - 测试空数据库
   - 验证表结构

6. **generate_report()** - 生成报告
   - 统计测试结果
   - 生成 Markdown 报告
   - 保存到 PHASE5_TEST_REPORT.md

1. **分层测试**: 从单元到集成，逐层验证
2. **自动化**: 所有测试可自动运行
3. **报告生成**: 自动生成测试报告

1. **测试数据隔离**: 使用独立的测试数据库
