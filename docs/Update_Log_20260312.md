## 新增功能

### 1. update_flag 数据筛选功能

#### 功能说明
在获取财务数据时，自动筛选 `update_flag`，优先使用修正后的数据版本，确保数据的准确性和时效性。

#### 核心逻辑
- `update_flag=0`: 初始发布版本（可能包含错误）
- `update_flag=1`: 更新/修正版本（推荐使用）
- 对于同一报告期，优先选择 `update_flag=1` 的记录，若无则使用 `update_flag=0`

#### 使用方式
无需额外配置，自动应用。程序会在日志中显示筛选信息：
```
update_flag 筛选: 原始 42 条 → 筛选后 21 条 (移除 21 条重复记录, 更新版 15 条, 初始版 6 条)
```

#### 修改文件
- `tushare_client.py`: 新增 `_filter_by_update_flag()` 方法，并在所有财务数据获取方法中自动调用

### 2. 资产负债表重构功能

#### 功能说明
将传统资产负债表重构为"资产-资本"结构，便于分析企业的金融资产、经营资产、有息债务和所有者权益。

#### 重构结构

```
资产结构
├── 金融资产合计
│   ├── 货币资金
│   ├── 交易性金融资产
│   ├── 债权投资
│   └── 投资性房地产等
├── 长期股权投资
└── 经营资产合计
    ├── 周转性经营投入合计
    │   ├── 营运资产小计
    │   └── 营运负债小计（减项）
    └── 长期经营资产合计

资本结构
├── 有息债务合计
│   ├── 短期债务
│   └── 长期债务
├── 所有者权益合计
│   ├── 归属于母公司股东权益
│   └── 少数股东权益
└── 资本总额
```

#### 核心计算公式

| 项目 | 计算规则 |
|------|---------|
| 其他应收款(调整后) | = 其他应收款 - 应收利息 - 应收股利 |
| 其他应付款(调整后) | = 其他应付款 - 应付利息 - 应付股利 |
| 周转性经营投入合计 | = 营运资产小计 - 营运负债小计 |
| 经营资产合计 | = 周转性经营投入合计 + 长期经营资产合计 |
| 有息债务合计 | = 短期债务 + 长期债务 |
| 资本总额 | = 所有者权益合计 + 有息债务合计 |

#### 使用方式

**方法1: 重构已有CSV文件**

### 3. 年报+TTM重构报表生成功能

#### 功能说明
生成过去10年年报 + 最新一期TTM的重构报表，便于进行长期财务趋势分析。

#### TTM计算规则

**定义**：TTM (Trailing Twelve Months) = 滚动12个月数据，用于消除季节性波动影响。

**适用范围**：

| 报表类型 | 数据性质 | TTM计算 |
|---------|---------|---------|
| 资产负债表 | 时点数据 | **不需要**，直接取最新季度数据 |
| 利润表 | 期间数据（累计值） | **需要**，计算滚动12个月 |
| 现金流量表 | 期间数据（累计值） | **需要**，计算滚动12个月 |

**计算公式**：

数据库数据特点：季度数据为累计值
```
Q1 = 1-3月累计
Q2 = 1-6月累计
Q3 = 1-9月累计
Q4 = 1-12月累计（年度数据）
```

TTM计算公式（以Q3为例）：
```
TTM = 今年Q3累计 - 去年Q3累计 + 去年Q4累计
    = 今年Q3 + 去年Q4单季度
    = 最近12个月滚动数据
```

示例（2025Q3-TTM）：
```
TTM = 2025Q3累计 - 2024Q3累计 + 2024Q4累计
    = (2025Q1+Q2+Q3) - (2024Q1+Q2+Q3) + (2024Q1+Q2+Q3+Q4)
    = 2024Q4 + 2025Q1+Q2+Q3
    = 最近12个月
```

**特殊情况**：
- 如果最新报告期是Q4（年报），则直接使用年报数据，不需要计算TTM

#### 使用方式

**方法1: 使用main.py命令行**
```bash
# 生成年报+TTM重构报表（默认10年）
python main.py 603345.SH --annual-ttm

# 指定年数
python main.py 603345.SH --annual-ttm --years 15

# 指定日期范围
python main.py 603345.SH --start-date 20100101 --annual-ttm
```

**方法2: 使用示例脚本**
```bash
python example_annual_ttm.py
```

**方法3: 编程调用**
