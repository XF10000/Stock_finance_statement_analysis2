# 利润表重构指南（股权价值增加表）

## 概述

本模块将传统利润表重构为"股权价值增加表"，计算息税前经营利润、息前税后利润、股权价值增加值等核心指标，便于分析企业的盈利能力和价值创造能力。

## 重构原理

### 传统利润表 vs 重构后的利润表

**传统结构**：
```
营业收入
- 营业成本
- 税金及附加
- 销售费用
- 管理费用
- 研发费用
- 财务费用
+ 投资收益
= 利润总额
- 所得税费用
= 净利润
```

**重构后结构（股权价值增加表）**：
```
息税前经营利润
- 经营利润所得税
= 息前税后经营利润

+ 息前税后金融资产收益
+ 长期股权投资收益
= 息税前利润总额

- 税后真实财务费用
= 净利润

- 股权资本成本
= 股权价值增加值
```

### 核心区别

1. **经营利润 vs 金融收益**
   - 经营利润：来自主营业务的利润（息税前经营利润）
   - 金融收益：来自金融资产投资的收益（利息收入、投资收益等）

2. **息前 vs 息后**
   - 息前：扣除财务费用之前的利润
   - 息后：扣除财务费用之后的利润（净利润）

3. **所得税分配**
   - 将所得税按利润来源进行分配
   - 分别计算经营利润、金融资产收益、财务费用各自承担的所得税

## 重构规则详细说明

### 结构设计

```
股权价值增加表
├── 营业收入
├── 营业成本
├── 营业成本率 = 营业成本 / 营业收入
├── 毛利 = 营业收入 - 营业成本
├── 毛利率 = 毛利 / 营业收入
├── 税金及附加
├── 营业税金及附加率 = 税金及附加 / 营业收入
├── 销售费用
├── 销售费用率 = 销售费用 / 营业收入
├── 管理费用
├── 管理费用率 = 管理费用 / 营业收入
├── 研发费用
├── 研发费用率 = 研发费用 / 营业收入
├── 资产减值损失
├── 信用减值损失
├── 资产减值损失率 = (资产减值损失 + 信用减值损失) / 营业收入
├── 资产处置收益
├── 加：其他收益
├── 加：营业外收入
├── 减：营业外支出
├── 营业外收支净额 = 营业外收入 - 营业外支出
├── 营业外收支净额率 = 营业外收支净额 / 营业收入
├── 息税前经营利润
├── 息税前经营利润率 = 息税前经营利润 / 营业收入
├── 实际所得税税率 = 所得税费用 / (税前利润 - 长期股权投资收益)
├── 经营利润所得税 = 息税前经营利润 × 实际所得税税率
├── 息前税后经营利润 = 息税前经营利润 - 经营利润所得税
├── 投资收益
│   ├── 对联营企业及合营企业的投资收益
│   └── 长期股权投资收益 = 对联营企业投资收益
├── 短期投资收益 = 投资收益 - 长期股权投资收益
├── (其中)利息收入
├── 净敞口套期收益
├── 公允价值变动收益
├── 汇兑收益
├── 其他综合收益的税后净额
├── 息税前金融资产收益
├── 金融资产收益所得税 = 息税前金融资产收益 × 实际所得税税率
├── 息前税后金融资产收益 = 息税前金融资产收益 - 金融资产收益所得税
├── 息税前利润总额 = 息前税后经营利润 + 息前税后金融资产收益 + 长期股权投资收益
├── 息前税后利润总额 = 息前税后经营利润 + 息前税后金融资产收益 + 长期股权投资收益
├── 财务费用
├── 真实财务费用 = 财务费用 + 利息收入
├── (其中)利息费用
├── 财务费用抵税效应 = 真实财务费用 × 实际所得税税率
├── 税后真实财务费用 = 真实财务费用 - 财务费用抵税效应
├── 税前利润 = 息税前利润总额 - 真实财务费用
├── 减：所得税费用
├── 净利润 = 税前利润 - 所得税费用
├── 净利润率 = 净利润 / 营业收入
├── 股权资本成本（默认8%）
└── 股权价值增加值 = 净利润 - 股权资本成本
```

### 核心计算公式

| 项目 | 计算规则 |
|------|---------|
| 毛利 | = 营业收入 - 营业成本 |
| 毛利率 | = 毛利 / 营业收入 |
| 营业外收支净额 | = 营业外收入 - 营业外支出 |
| **息税前经营利润** | = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 资产减值损失 - 信用减值损失 + 资产处置收益 + 其他收益 + 营业外收入 - 营业外支出 |
| **实际所得税税率** | = 所得税费用 / (税前利润 - 长期股权投资收益) |
| **经营利润所得税** | = 息税前经营利润 × 实际所得税税率 |
| 息前税后经营利润 | = 息税前经营利润 - 经营利润所得税 |
| 长期股权投资收益 | = 对联营企业及合营企业的投资收益 |
| 短期投资收益 | = 投资收益 - 长期股权投资收益 |
| **息税前金融资产收益** | = 短期投资收益 + 利息收入 + 净敞口套期收益 + 公允价值变动收益 + 汇兑收益 + 其他综合收益 |
| **金融资产收益所得税** | = 息税前金融资产收益 × 实际所得税税率 |
| 息前税后金融资产收益 | = 息税前金融资产收益 - 金融资产收益所得税 |
| 息税前利润总额 | = 息前税后经营利润 + 息前税后金融资产收益 + 长期股权投资收益 |
| **真实财务费用** | = 财务费用 + 利息收入 |
| **财务费用抵税效应** | = 真实财务费用 × 实际所得税税率 |
| 税后真实财务费用 | = 真实财务费用 - 财务费用抵税效应 |
| 税前利润 | = 息税前利润总额 - 真实财务费用 |
| 净利润 | = 税前利润 - 所得税费用 |
| 股权资本成本 | 默认值：8%（可在配置文件中调整） |
| 股权价值增加值 | = 净利润 - 股权资本成本 |

### 所得税项目计算说明

**设计原理**：将所得税按利润来源进行分配，分别计算经营利润、金融资产收益、财务费用各自承担的所得税影响。

| 项目 | 公式 | 说明 |
|------|------|------|
| **实际所得税税率** | = 所得税费用 / (税前利润 - 长期股权投资收益) | 排除长期股权投资收益（通常适用不同税率）后的实际税率 |
| **经营利润所得税** | = 息税前经营利润 × 实际所得税税率 | 经营活动产生的利润应承担的所得税 |
| **金融资产收益所得税** | = 息税前金融资产收益 × 实际所得税税率 | 金融资产投资收益应承担的所得税 |
| **财务费用抵税效应** | = 真实财务费用 × 实际所得税税率 | 利息支出可抵税，产生的抵税收益 |

**计算顺序**：
1. 先计算实际所得税税率
2. 分别计算三个所得税相关项目
3. 用所得税项目计算税后利润

**示例**（假设数据）：
```
息税前经营利润 = 100亿
息税前金融资产收益 = 10亿
真实财务费用 = 5亿
所得税费用 = 20亿
长期股权投资收益 = 2亿
税前利润 = 105亿（= 100 + 10 + 2 - 5 = 107，但实际从报表读取）

实际所得税税率 = 20 / (105 - 2) = 19.42%

经营利润所得税 = 100 × 19.42% = 19.42亿
金融资产收益所得税 = 10 × 19.42% = 1.94亿
财务费用抵税效应 = 5 × 19.42% = 0.97亿

息前税后经营利润 = 100 - 19.42 = 80.58亿
息前税后金融资产收益 = 10 - 1.94 = 8.06亿
税后真实财务费用 = 5 - 0.97 = 4.03亿

净利润 = 80.58 + 8.06 + 2 - 4.03 = 86.61亿
或：净利润 = 105 - 20 = 85亿（从报表直接读取）
```

### 比率类指标说明

所有比率类指标均以营业收入为分母，便于分析各项费用和利润占营业收入的比例：

| 比率指标 | 公式 |
|---------|------|
| 营业成本率 | = 营业成本 / 营业收入 |
| 毛利率 | = 毛利 / 营业收入 |
| 营业税金及附加率 | = 税金及附加 / 营业收入 |
| 销售费用率 | = 销售费用 / 营业收入 |
| 管理费用率 | = 管理费用 / 营业收入 |
| 研发费用率 | = 研发费用 / 营业收入 |
| 资产减值损失率 | = (资产减值损失 + 信用减值损失) / 营业收入 |
| 营业外收支净额率 | = 营业外收支净额 / 营业收入 |
| 息税前经营利润率 | = 息税前经营利润 / 营业收入 |
| 净利润率 | = 净利润 / 营业收入 |

## 使用方法

### 1. 基本使用

```python
import pandas as pd
from income_statement_restructure import restructure_income_statement

# 读取利润表数据（转置格式）
df = pd.read_csv('data/603345.SH_income.csv', encoding='utf-8-sig')

# 重构利润表
df_restructured = restructure_income_statement(df)

# 保存结果
df_restructured.to_csv('data/603345.SH_income_restructured.csv', 
                       index=False, encoding='utf-8-sig')
```

### 2. 与TushareClient集成

```python
from tushare_client import TushareClient
from income_statement_restructure import restructure_income_statement

# 初始化客户端
client = TushareClient()

# 获取原始利润表
income = client.get_income('603345.SH', translate=True)

# 重构利润表
income_restructured = restructure_income_statement(income)

# 保存
client.save_to_csv({'income_restructured': income_restructured}, 
                   '603345.SH')
```

## 字段说明

### 经营利润相关字段

| 项目 | 说明 |
|------|------|
| 息税前经营利润 | 扣除所有经营费用后、扣除财务费用和所得税之前的经营利润 |
| 经营利润所得税 | 经营利润应承担的所得税 |
| 息前税后经营利润 | 扣除所得税后的经营利润，反映经营活动的真实盈利能力 |

### 金融收益相关字段

| 项目 | 说明 |
|------|------|
| 短期投资收益 | 短期金融资产投资产生的收益 |
| 息税前金融资产收益 | 所有金融资产产生的收益总和（税前） |
| 金融资产收益所得税 | 金融资产收益应承担的所得税 |
| 息前税后金融资产收益 | 扣除所得税后的金融资产收益 |

### 财务费用相关字段

| 项目 | 说明 |
|------|------|
| 真实财务费用 | 加回利息收入后的真实财务费用 |
| 财务费用抵税效应 | 财务费用产生的税收抵扣效应 |
| 税后真实财务费用 | 扣除抵税效应后的真实财务费用 |

### 特殊调整项

1. **利息收入调整**
   ```
   真实财务费用 = 财务费用 + 利息收入
   ```
   - 财务费用中已扣除利息收入，需要加回以反映真实的财务成本

2. **长期股权投资收益**
   ```
   长期股权投资收益 = 对联营企业及合营企业的投资收益
   ```
   - 长期股权投资收益通常适用不同的税率，需要单独处理

## 分析应用

### 1. 盈利能力分析

```python
# 计算各项利润率
营业收入 = df_restructured[df_restructured['项目'] == '营业收入'][date].values[0]
毛利率 = df_restructured[df_restructured['项目'] == '毛利率'][date].values[0]
息税前经营利润率 = df_restructured[df_restructured['项目'] == '息税前经营利润率'][date].values[0]
净利润率 = df_restructured[df_restructured['项目'] == '净利润率'][date].values[0]

print(f"毛利率: {毛利率*100:.2f}%")
print(f"息税前经营利润率: {息税前经营利润率*100:.2f}%")
print(f"净利润率: {净利润率*100:.2f}%")
```

### 2. 费用结构分析

```python
# 计算各项费用率
销售费用率 = df_restructured[df_restructured['项目'] == '销售费用率'][date].values[0]
管理费用率 = df_restructured[df_restructured['项目'] == '管理费用率'][date].values[0]
研发费用率 = df_restructured[df_restructured['项目'] == '研发费用率'][date].values[0]

print(f"销售费用率: {销售费用率*100:.2f}%")
print(f"管理费用率: {管理费用率*100:.2f}%")
print(f"研发费用率: {研发费用率*100:.2f}%")
```

### 3. 价值创造分析

```python
# 计算股权价值增加值
净利润 = df_restructured[df_restructured['项目'] == '净利润'][date].values[0]
股权价值增加值 = df_restructured[df_restructured['项目'] == '股权价值增加值'][date].values[0]

if 股权价值增加值 > 0:
    print(f"企业创造了股权价值: {股权价值增加值:,.2f}")
else:
    print(f"企业未能创造股权价值，价值损失: {-股权价值增加值:,.2f}")
```

## 注意事项

1. **数据格式要求**
   - 输入数据必须是转置格式（字段名为行，日期为列）
   - 字段名可以是中文或英文，模块会自动识别

2. **缺失值处理**
   - 缺失值会被自动填充为0进行计算
   - 建议在使用前检查数据完整性

3. **所得税税率计算**
   - 实际所得税税率排除了长期股权投资收益的影响
   - 如果税前利润等于长期股权投资收益，税率会设为0

4. **股权资本成本**
   - 默认值为8%，可在配置文件中调整
   - 股权资本成本 = 所有者权益合计 × 8%
